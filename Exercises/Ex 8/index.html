Principais Formatações de Textos

Negrito / Destaque

✗ → Nesta frase temos um termo em negrito usando a tag <b> </b> (não semântica).

✓ → Nesta frase, tenho um termo em destaque usando a tag <strong> </strong> (semântica).

Itálico / Ênfase

✗ → Nesta frase temos um termo em itálico usando a tag <i> </i> (não semântica).

✓ → Nesta frase, tenho um termo em ênfase usando a tag <em> </em> (semântica).

Marcador de Texto

✓ → Nesta frase temos um termo em marcado usando a tag <mark> </mark>.

Texto Grande e Pequeno

🤷 → Estamos a criar um texto grande (não utilizar o <big> </big>) e um outro texto pequeno (pode ser utilizado o <small> </small>).

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado.

Texto Inserido

✓ → Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois.

✗ → Existe também o sublinhado com a tag <u> </u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup> </sup>.

Texto Subscrito

Para inserir coisas do tipo H2O com a tag <sub> </sub>.
